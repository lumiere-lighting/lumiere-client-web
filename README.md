# Lumiere Client (Web)

A website to interact with the Lumiere service.

## Development

Run development server with:

```bash
npm run dev
```

## Build

To build static site:

```bash
npm run build
```

## Deploy

Deployed to Github Pages.  The following will build and publish site.

```bash
npm run deploy
```
